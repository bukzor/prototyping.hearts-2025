# Rules Collection

Hearts game rules. Each file describes one rule or rule variant.

## What Belongs Here

- Standard Hearts rules (one per file)
- Optional variant rules (marked in frontmatter)
- Rule clarifications and edge cases

## What Does NOT Belong

- Strategy guides
- UI specifications
- Implementation details

## Frontmatter

Optional. When used:
```yaml
optional: true|false  # Is this rule a variant/house rule?
```

## Reference

Standard rules per Wikipedia: https://en.wikipedia.org/wiki/Hearts_(card_game)
