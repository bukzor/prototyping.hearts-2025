# Milestones Collection

Development milestones for Hearts Online.

## Naming Convention

- `aNNN-$SLUG` - alpha (family only)
- `bNNN-$SLUG` - beta (friends)
- `vNNN-$SLUG` - stable (public)

Zero-pad to 3 digits. Slug is kebab-case.

## What Belongs Here

- Milestone definitions (scope, goals, success criteria)
- What's in/out for each milestone

## What Does NOT Belong

- Detailed task lists (use issue tracker)
- Time estimates
- Status updates (use git history)

## Frontmatter

Optional. When used:
```yaml
status: planned|active|complete
```
