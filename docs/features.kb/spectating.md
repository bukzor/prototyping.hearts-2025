# Spectating (Kibitz)

Watch games in progress.

## Configuration

- Enabled/disabled at table host discretion
- "Enable kibitz" toggle when creating table

## Behavior

- Spectators see all played cards
- Spectators do NOT see unplayed hands (prevents cheating via screen share)
- Optional: delay spectator view by N seconds

## Notes

Classic term "kibitz" from card game culture.
