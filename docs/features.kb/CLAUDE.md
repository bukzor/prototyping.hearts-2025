# Features Collection

Features to build for Hearts Online. Each file describes one feature.

## What Belongs Here

- User-facing features (matchmaking, leaderboards, chat)
- System features (time controls, spectating)
- Feature requirements and acceptance criteria

## What Does NOT Belong

- Implementation details (those go in code or ADRs)
- Bug reports
- Milestone planning (see milestones.kb/)

## Frontmatter

Optional. When used:

```yaml
priority: high|medium|low
milestone: aNNN-slug # Target milestone
```
